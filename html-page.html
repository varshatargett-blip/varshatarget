What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create and structure content on the web. HTML describes the structure of a web page using a series of elements.

HTML elements tell the browser how to display the content. They are represented by tags, written using angle brackets like <tagname>.

💡 Did you know? HTML was first created by Tim Berners-Lee in 1991. The latest version is HTML5, which introduced powerful features like canvas, audio, video, and semantic elements.
  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", etc.

HTML Document Structure

Every HTML document follows a basic structure. Here is a simple example of an HTML document:

HTML 
<!-- Basic HTML Document Structure -->
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first paragraph.</p>
  </body>
</html>

Let's break down each part:

  • <!DOCTYPE html> — Declares the document type as HTML5
  • <html> — The root element of the page
  • <head> — Contains meta information about the page
  • <title> — Sets the browser tab title
  • <body> — Contains the visible page content

How HTML Works

When you open a web page in your browser, the browser reads the HTML file and renders it visually. The browser does not display the HTML tags themselves, but uses them to determine how to display the content.

🌐 HTML defines the structure and content of a page. CSS handles styling and layout. JavaScript adds interactivity. Together they form the three core technologies of the web.

HTML files are plain text files saved with the .html or .htm extension. You can create an HTML file using any simple text editor such as Notepad (Windows) or VS Code.
